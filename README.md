# **Monitoring temperatures and spO2 using an Arduino Uno and ESP32 on OHmnibot**

## ***Prequisite setting:***
Hardwares:

- Arduino Uno 
- NodeMCU ESP32
- Contactless IR temperature sensor (GY-906 MX90614 or others)
- MAX30100 Pulse Oximeter Sensor 

Softwares:

- Rosserial package
- HiveMQ
